# harvard-artifacts-analytics
Streamlit + SQL + ETL project for Harvard Artifacts API
This is my first project!  
In this project, I fetch artifact data from the **Harvard Art Museums API** and store it in **MySQL**. I also use **Python**, **Jupyter Notebook**, and **Streamlit** to explore and show the data.

---

## What this project does

- **Collect data:** Get artifacts from the Harvard API  
- **Store data:** Save it in a MySQL database 

- **Analyze data:** Run queries to learn about the artifacts  
- **Show data:** Use Streamlit to display artifact info, images, and colors
